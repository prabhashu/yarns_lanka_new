# Yarns Lanka - Premium Textile Excellence

Welcome to the official repository for **Yarns Lanka**, a modern web application showcasing the finest Sri Lankan textile products, fabrics, and artisanal craftsmanship.

## 🚀 Getting Started

If you are setting up this project on a new computer, follow these simple steps to get the development environment ready.

### 1. Prerequisites
Ensure you have **Node.js** (v18.x or later) installed on your system. You can check your version by running:
```bash
node -v
```

### 2. Installation
First, clone the repository or download the source code. Then, open your terminal in the project directory and install the necessary dependencies:

```bash
npm install
```

### 3. Running the Development Server
Once the installation is complete, start the local development server:

```bash
npm run dev
```

### 4. Viewing the Project
Open your browser and navigate to:
[http://localhost:3000](http://localhost:3000)

The application will automatically update as you make changes to the source files.

## � Pushing Updates to GitHub

To save your work and push new updates to the GitHub repository, follow these steps in your terminal:

```bash
# 1. Stage all your changes
git add .

# 2. Commit the changes with a descriptive message
git commit -m "Describe your changes here"

# 3. Push the updates to the repository (assuming 'main' is your branch)
git push origin main
```
